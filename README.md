Веб-додаток для перетворення числових значень у суму в українських гривнях прописом. Додаток дозволяє користувачу ввести число, натиснути кнопку та отримати текстове представлення суми відповідно до правил української граматики. Результат автоматично копіюється в буфер обміну, відображається на сторінці, а поле введення очищується.
